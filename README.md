# Member Operations Dashboard

An Angular + .NET portfolio project demonstrating enterprise staff tool patterns for credit union operations.

## Tech Stack

- **Frontend**: Angular 18 (standalone components) + Angular Material
- **Backend**: ASP.NET Core 8 Web API
- **Database**: PostgreSQL 16
- **Authentication**: JWT

## Setup Instructions

### Prerequisites
- Node.js 20+
- .NET 8 SDK
- Docker Desktop (for PostgreSQL)
